\section{Theoretical Background: Phase Portraits}
In general, a phase portrait displays the evolution of the state of a dynamical system by plotting subsequent states in the same axis.
For one-dimensional continuous systems the axes would usually be $\dot{x}$ and $x$.
For more than one dimension or state variable one has to chose them to be well representable in two dimensions.
In the examined system here we have an iterated map with two dimensions which are also the axes in the phase portrait plot.

If an attractive fixed point exists in the system, later iterations would be ever closer than previous iterations.
A stable orbit of length $n$ could hardly be distinguished from $n$ fixed points being present and would be needed to be further examined by the other methods employed in this report.
Chaotic conditions can be inferred when there is no discernible pattern in the phase portrait.
