\subsection{Bifurcation cascade theory}

A bifurcation refers to a qualitative change in the behavior of a dynamical system as a parameter is varied. In the context of our system we are interested in how the number of stable orbits reacts to varying one of the r parameters while keeping other parameters constant. At low r values the system has only one stable orbit. At a certain r value this stable orbit will split in to two undergoing a period doubling bifurcation. A r is increased more these orbits will move further away from eachother and eventually undergo another period doubling. The step in r value between these bifurcations will keep getting smaller with each bifurcation. After reaching a certain r value the number of stable orbits will approach infinity and the system can be said to be chaotic.
