\subsection{Bifurcation Cascade}
Figure \ref{fig:bifurcationcascade} shows the bifurcation cascade of the coupled logistic map. At the lowest $r_2$ values, from 2.8 to around 2.87, 
we can see one branch. From $r_2 \approx 2.85$ the branch starts splitting into two and these branches separate from each other. At $r_2 \approx 3.7$ 
both branches split again so that we have four different branches. Each branch splits again at $r_2 \approx 3.9$. At slightly higher $r_2$ values the 
branches start splitting faster so that we end up with a point cloud that grows larger with higher $r_2$ values.
